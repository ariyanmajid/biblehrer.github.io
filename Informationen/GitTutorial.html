Git Tutorial

  • Install Git
  • git clone "url des Repos"
  • git config user.email= "newEmail@gmail.com"
  • Make changes
  • "git add *" This adds all changes
  • "git add /file.html" This adds only changes to file.html
  • "git commit -m "Comment of your changes"
  • "git push
  • ! Always use git pull before doing anything to an existing repo
